# shared

Cross-workspace design tokens, components, types, and utilities. Consumed by `admin`, `portal`, `site` (presentation shell), and `demos`.

Subdirectories:
- `components/` — UI primitives (buttons, form controls, badges, modals, tables, cards, status indicators).
- `types/` — TypeScript types shared across worker + frontends (entity shapes, API request/response contracts).
- `utils/` — formatting, validation, money math helpers.

**Status:** scaffolded but empty. Files land as admin/portal/presentation are built out.
